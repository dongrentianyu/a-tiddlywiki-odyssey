# 移动端同步

- Download: https://github.com/tiddly-gittly/TidGi-Mobile/releases/latest
- Code: https://github.com/tiddly-gittly/TidGi-Mobile
- Site: https://tiddly-gittly.github.io/TidGi-Official-Website/#TidGi-Mobile:TidGi-Mobile

太记移动端是开源软件，配套太记桌面端使用。你可以从上面 github 网址中下载软件。如果你从谷歌软件商店下载，可能会收取费用。因为谷歌软件商店上架软件需要开发者交纳费用。同理，ios 平台也会收取费用。所以更建议从 github 上面下载。

其次，太记移动端并不能直接与 github 同步。太记移动端需要先和桌面端保持同步，桌面端再和 github 仓库进行同步。
