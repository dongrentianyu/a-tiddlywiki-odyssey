# 安装太记

## 为什么使用太记？

- 太记是全平台软件，有安卓端配套软件。
- 太记使用的是 nodejs 存储方式，可使用 git 进行版本管理。
- 太记可以同时使用多个 tiddlywiki。
- 太记可以与 github desktop 进行自动同步。
- 太记预置了很多插件，让新手更容易上手使用。
- 太记还内置了新手引导等网站，可供用户查找资料。
- 太记是国内开发者开发，开源免费，更新频繁，反馈问题能及时解决。

tiddlywiki 有很多种存储方式。可以分为 html 单文件版本和 nodejs 多文件版本。可以在浏览器内打开，也可以用软件打开，也可以自己配置 nodejs 版本打开。还可以借助通用网站打开。但这么多种方式反而会让新手困惑，不知道从什么地方开始。所以本文档主要推荐太记。本人也基本上只用太记这种形式了。在安装与备份方面，都会围绕太记来说明，而其他方式则会在社区版块介绍。

之前在 QQ 群里也有过讨论，有些人认为应该给新手推荐原生的太微，以为原生太微就足以满足大部分需求，省下的只需要安装几个插件就可以了。而林一二则认为应该给新手推荐配置相对完整的太微，这样可以让新手省去不少时间，也可以达到先上手使用，后面再摸索太微的目的。当然预置插件过多，也容易导致新手迷糊，不知道怎么使用。

当时讨论十分激烈，但总的来说，我更认同林一二的观点，即新手更应该先使用配置好的太微，后续再摸索太微一段时间后，配置出自己的太微。这里说的新手是没有技术背景的小白，如果是有技术背景的，那么不妨自行探索太微。

太记是全平台软件，也有配套的安卓移动端软件。ios 端目前没有。太记是由国内开发者开发的，这一点比其他软件平台要好很多，至少出了问题可以在 QQ 群或者 github 上反馈，能及时得到修复。

## 如何安装太记？

目前太记最新版本可以去上面 github 里下载。如果不能登录 github，那就去太微 QQ 群里说一句。一般最新版本发布，林一二都会往群里上传最新版本的太记。如果林一二没有在群里上传的话，也可以去资源群里发私信。

最新版本太记不支持 win7，也不支持 32 位的 windows 系统。

如果是从上面官网下载的话，有可能会下载坏的安装包。解决办法还是去 QQ 群里找林一二，询求最新版本太记安装包。
