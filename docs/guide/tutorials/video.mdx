# 视频资源

太微的视频资源非常稀少。这里也只是列举一些常见的，少部分单个视频的可能就不放上来了。需要有系统的教程或者比较多的内容就放上来。太微资源本身也不多，随便在浏览器或者平台搜索一下，就能发现相关的内容。如果有人想要制作太微相关的视频教程，欢迎联系我，别的不会，但至少我肯定会过去点赞支持。
