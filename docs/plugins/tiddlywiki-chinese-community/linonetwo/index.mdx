# 林一二插件库

林一二本身没有搞一个插件库。但他开发了很多插件。所以我认为有必要以插件库的形式进行专题介绍。他开发的插件大部分都放在太微中文社区里，通过 CPL 就可以搜索安装。

并且特别推荐使用 CPL 安装，因为林一二开发的很多插件，可能都有依赖其他插件。如果你只是从演示网站里拖动插件到你的太微，很可能就会因为缺失其他依赖插件而无法正常使用。

林一二有一部分开发的插件是为太记服务的。即太记在启动时会自动为使用的太微安装上。因为不安装上可能就无法正常启动。这些插件不需要太深入了解。知道有这个东西就可以了。然后有一些插件，太记模板会默认安装上了。

之前林一二可能是把插件都放在一个仓库里，而如今都拆分开来了。我觉得这样挺好的。
