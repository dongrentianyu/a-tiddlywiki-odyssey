## 平台与发行版的区别

平台是指你使用的设备与软件。因为软件是与对应的设备挂钩的。比如有些软件能在 windows 电脑下使用，但没有 mac 版本。而太微本身是有两种形态的，一种是 html 保存形态，一种是 nodejs 保存形态。有些软件两者都支持。有些软件则只支持一种。像太记就只支持 nodejs 保存形态。使用 html 导入后，还是会解析成 nodejs 文件夹形式，而非以 html 形式保存。

而发行版则是说，你可以把任何一个模板保存为 html 文件。然后在任何平台上使用。即你可以在太记上使用 tiddlywiki-xp 版本。你也可以把太记的模板保存一份下来，在其他平台上使用。
